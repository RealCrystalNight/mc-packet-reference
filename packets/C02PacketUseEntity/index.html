Notes

ATTACK sends damage to the target entity. INTERACT opens inventory/rides. INTERACT_AT is for hitting specific entity hitboxes (like Armor Stand limbs).

implementation Implementation Cases

C02PacketUseEntity is the packet sent when a player interacts with an entity — either attacking it (ACTION ATTACK) or right-clicking it (ACTION INTERACT). The packet carries the target entity ID and the action type. This is the single most important packet for combat modules. KillAura spams it every tick on the target. Criticals injects C04 position offset packets before it. NoDamageFriends cancels it if the target is a friend. AntiFireball attacks fireball projectiles with it. AutoClicker sends it on mouse-over entities. The packet flows through NetHandlerPlayServer.processUseEntity() on the server side, which dispatches to Entity.attackEntity() or interact().

KillAura / MultiAura / SwitchAura (Ketamine v5.7, Raze Recode 2.0, Slice v15, Mint B1, Gugustus, Koks v1, Swift Developer Alpha, Tenacity 4.0, Lycoris 2.0, Nekoware v1, Jigsaw 0.24)

Automatically attacks the nearest/selected entity every tick. MultiAura attacks multiple entities. SwitchAura cycles targets.

Each tick, the module finds the best target (closest, lowest health, etc.), calls mc.thePlayer.swingItem() for the visual arm swing, then sends C02PacketUseEntity with ATTACK action. Most implementations also follow the attack with autoblock (C07/C08) and criticals (C04 offset). Advanced auras also send INTERACT with an empty hand before attacks on some servers.

// Ketamine v5.7 - Aura.java this.mc.thePlayer.sendQueue.sendPacket(new C02PacketUseEntity(target, C02PacketUseEntity.Action.ATTACK)); // Raze Recode 2.0 - Aura.java mc.getNetHandler().addToSendQueue(new C02PacketUseEntity(entity, C02PacketUseEntity.Action.ATTACK)); // Jigsaw 0.24 - KillAura.java sendPacket(new C02PacketUseEntity(en, Action.ATTACK)); // Full KillAura loop example: public void onTick() { EntityLivingBase target = findTarget(); if (target == null) return; mc.thePlayer.swingItem(); // Send attack mc.thePlayer.sendQueue.addToSendQueue(new C02PacketUseEntity(target, C02PacketUseEntity.Action.ATTACK)); // Also send criticals if on ground if (mc.thePlayer.onGround && doCriticals) sendCritPackets(); // Autoblock after attack if (autoBlock) mc.thePlayer.sendQueue.addToSendQueue( new C07PacketPlayerDigging(C07PacketPlayerDigging.Action.RELEASE_USE_ITEM, BlockPos.ORIGIN, EnumFacing.DOWN)); }
Vanilla hook: NetHandlerPlayServer.processUseEntity() on server side. Client side: PlayerControllerMP.attackEntity() calls sendPacket(). For interception use EventSendPacket.
Criticals (critical hit injection) (Lycoris 1.0, Mint B1, Vestige 3.0, Jigsaw 0.24)

Forces critical hit particles and damage by sending position offset packets before every attack.

When C02PacketUseEntity with ATTACK is detected in EventSendPacket, the module cancels it temporarily. It first sends 2-3 C04PacketPlayerPosition packets with small Y offsets (0.0625, 0) to trick the server into thinking the player fell and landed (triggering the critical hit check). Then it sends the original C02. The entire sequence happens in the same tick.

// Lycoris 1.0 - Criticals.java @EventTarget public void onSend(EventSendPacket e) { if (e.getPacket() instanceof C02PacketUseEntity) { C02PacketUseEntity packet = (C02PacketUseEntity) e.getPacket(); if (packet.getAction() == C02PacketUseEntity.Action.ATTACK && mc.thePlayer.isCollidedVertically && hurtTimeCheck()) { // Inject critical positions doCrits(); // The attack packet goes through after criticals } } } private void doCrits() { double x = mc.thePlayer.posX, y = mc.thePlayer.posY, z = mc.thePlayer.posZ; mc.getNetHandler().addToSendQueue(new C04PacketPlayerPosition(x, y + 0.0625, z, false)); mc.getNetHandler().addToSendQueue(new C04PacketPlayerPosition(x, y, z, false)); // The attack C02 follows immediately }
Vanilla hook: EventSendPacket — detect C02 ATTACK, inject C04 packets BEFORE the C02 reaches the server. The key is packet ordering.
NoDamageFriends / TeamCheck (Celestial Premium 6.0 (NoDamageFriends))

Prevents attacking players on your friends list or team.

In EventSendPacket, if the packet is C02 ATTACK, the module gets the target entity from the packet and checks if the target"s name is in the friends list. If so, the event is cancelled.

// Celestial Premium 6.0 - NoDamageFriends.java @EventTarget public void onSendPacket(EventSendPacket event) { if (event.getPacket() instanceof C02PacketUseEntity) { C02PacketUseEntity packet = (C02PacketUseEntity) event.getPacket(); if (packet.getAction() == C02PacketUseEntity.Action.ATTACK) { Entity target = packet.getEntityFromWorld(mc.theWorld); if (target != null && Celestial.instance.friendManager.isFriend(target.getName())) { event.setCancelled(true); // don"t attack friend } } } }
Vanilla hook: EventSendPacket — check instanceof C02PacketUseEntity.
AntiFireball (Gugustus)

Automatically attacks fireball entities (Ghast fireballs, Dragon fireballs).

The module listens for S0EPacketSpawnObject. When the spawned object is type 63 (fireball) or 64 (small fireball), it tracks the entity. The KillAura then targets these fireball entities and sends C02 ATTACK at them to deflect/destroy them.

// Gugustus - AntiFireBall (combined with KillAura) // First, detect fireball spawn: if (packet instanceof S0EPacketSpawnObject) { S0EPacketSpawnObject s0e = (S0EPacketSpawnObject) packet; if (s0e.func_148993_l() == 63 || s0e.func_148993_l() == 64) { // Track fireball entity, add to attack list } } // Then in KillAura tick: mc.thePlayer.sendQueue.addToSendQueue(new C02PacketUseEntity(fireballEntity, C02PacketUseEntity.Action.ATTACK));
Vanilla hook: S0EPacketSpawnObject + EventSendPacket (C02).
AutoClicker (Astro 2.2.1)

Automatically clicks on entities when the mouse is held down.

While the user holds left-click, the module sends C02 ATTACK packets at CPS rate (clicks per second). It also sends the swing animation (C0APacketAnimation) before each attack.

// Astro 2.2.1 - AutoClicker.java if (mc.gameSettings.keyBindAttack.isKeyDown() && mc.objectMouseOver.entityHit != null) { if (cpsTimer.delay(1000 / cps)) { mc.thePlayer.swingItem(); mc.getNetHandler().getNetworkManager().sendPacketNoEvent( new C02PacketUseEntity(mc.objectMouseOver.entityHit, C02PacketUseEntity.Action.ATTACK)); cpsTimer.reset(); } }
Vanilla hook: Game loop tick → check mouse state → send C02 if conditions met.
Client send path: PlayerControllerMP.attackEntity(EntityPlayerSP, Entity) → mc.player.sendQueue.addToSendQueue(new C02PacketUseEntity(target, ATTACK)) → EventSendPacket → NetworkManager.sendPacket() → Netty → Server. Hook points: 1. EventSendPacket — Most popular. Check instanceof C02PacketUseEntity, read getAction(). 2. PlayerControllerMP.attackEntity() — Override to intercept before packet creation. 3. NetworkManager.sendPacket() — Global packet intercept. Server receive: NetHandlerPlayServer.processUseEntity() → entity.attackEntityFrom(DamageSource.causePlayerDamage(player), damage) for ATTACK, or entity.interact() for INTERACT.
